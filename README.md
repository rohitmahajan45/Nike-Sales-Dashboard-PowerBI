# Nike Sales Dashboard (Power BI)

## Project Overview
This interactive Power BI dashboard analyzes Nike sales performance across different regions, states, cities, and sales methods.

## Dashboard Features
- Total Profit
- Units Sold
- Number of Retailers
- Profit by Cities
- Sales by State
- Region Filter
- Sales Method Filter
- Year & Quarter Slicer
- Interactive Visualizations

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling
